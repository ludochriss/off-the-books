Concepts

Off-The-Books exchange order book

The order book is an off exchange location for orders to be placed. Exchanges and large players in the space use order book data to trade against market participants by selling order book information to other financial players. This is designed to protect against that on-sell of information.

The order book is a simple unstructued data table that holds the orders in a list and is updated in real time as orders are placed and removed.

This will be used to trigger orders as a stop loss or take profit for Binance Spot orders as binance does not allow this funcitonality for spot markets.

Trading View alert based trades

The Trading view alert based order trigger allows integration with tradingviews charting software to leverage the alert system to trigger orders from technical analysis rather than a single price level, Orders can be placed from gradients and also conditions such as moving average crosses, as well as any other condition that can trigger an alert.

Resources

Azure Functions are used as a back end APIs which depend on a library of code. Severless architecture minimizes the hosting cost to about $1 a month. The Apis are wrapped in an api management layer to provide a single point of access and to provide a security layer.

Currently using Table storage as a data layer with mongo on roadmap.
